# AI Searches Visualization

This repository contains Python implementations of Informed search algorithms with Tkinter-based GUI visualizations, designed for an AI class project.

## Algorithms
- **Greedy Best First Search (BFS)**: Explores the cheapest h(val).
- **A star (A*)**: Explores the cheapest option using h(n)+g(n).


## Requirements
- Python 3.x
- Tkinter (included with standard Python)

## Installation
1. Clone the repository:
   ```
   https://github.com/mAbdullah02-bit/Informed-Searches-Visualization.git
   ```
2. Run File:
   ```
   py filename.py
   ```

## Usage
- Run a Python file (e.g., `python bfs.py`).
- Select Algorithm and click "Run" to see the algorithm’s execution.
